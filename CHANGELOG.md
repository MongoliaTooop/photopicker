# 更新日志

## v3.2 - 2026-07-22

### 新增功能

- **缩略图尺寸预设系统**：打开文件夹时自动检测已有缩略图，无缩略图则弹出设置对话框
  - 快速加载（400px）— 极小缩略图，秒开
  - 平衡（800px）— 推荐，够用且流畅
  - 清晰预览（1200px）— 较大缩略图，看清更多细节
  - 自定义尺寸 — 输入长边像素值（200-4000px）
  - 不压缩 — 使用原图，不生成缩略图（大文件夹可能较卡）
- **缩略图文件名带尺寸标记**：如 `IMG_001_800.jpg`，不同尺寸不冲突，旧格式（800px无后缀）自动兼容

### 界面调整

- **「匹配RAW」按钮样式改为绿色**：与右侧「选中」按钮同色，更醒目
- 缩略图设置对话框 UI：2×2 预设卡片 + 自定义输入 + 确认按钮

## v3.1 - 2026-07-22

### 修复

- **移除工具栏多余「导出」按钮**：原工具栏「导出」按钮引用了未定义的 `openExportDialog()` 函数，点击会报错。匹配RAW对话框内已有完整导出功能，工具栏按钮属于重复入口，现已移除。

### 功能调整

- **匹配RAW对话框导出选项重新定义**：
  - 「复制到文件夹」改名为「复制RAW到文件夹」，语义更明确
  - 新增「复制JPG+RAW到文件夹」选项，同时复制选中JPG和对应RAW到目标文件夹，JPG和RAW分别存放在 `JPG/` 和 `RAW/` 子目录中
  - 「导出清单（CSV）」保持不变

## v2.0 - 2026-07-21

### 新增功能

- **缩略图压缩生成**：打开文件夹时自动在子目录「缩略图」中生成压缩缩略图（长边800px，JPEG质量0.7），左侧列表使用缩略图替代原图，大幅提升加载速度。已有缩略图自动复用，不重复生成。
- **预览压缩设置**：工具栏新增「⚙ 设置」按钮，可调节预览主图的压缩比（1-100%）。低于100%时预览图按比例压缩显示，右上角出现「查看原图」按钮，点击可全屏查看未压缩原图。设置自动保存，下次打开自动恢复。
- **分页加载**：缩略图列表每次加载100张，滚动到剩余约15张时自动预加载下一批100张，支持上千张照片流畅浏览。
- **文件体积信息**：右侧信息面板新增「图片体积」和「缩略图体积」两个字段，直观展示压缩效果。

### 优化

- 打开文件夹时请求读写权限（用于生成缩略图子文件夹）
- 缩略图后台批量生成，不阻塞选片操作
- 键盘导航到未加载的照片时自动触发分页加载
- 预览压缩URL自动回收，防止内存泄漏

## v1.0 - 2026-07-08

### 初始版本

- **JPG选片界面**：暗色主题三栏布局（缩略图列表 + 大图预览 + 文件信息面板）
- **三态标记系统**：选中（绿色）/ 待定（黄色）/ 删除（红色），全键盘快捷键操作
- **RAW文件智能匹配**：根据JPG文件名自动匹配对应RAW文件（支持CR2/CR3/NEF/ARW/RAF/DNG等），匹配结果预览
- **三种导出方式**：复制到文件夹、导出CSV清单
- **EXIF信息读取**：相机、镜头、焦距、光圈、快门、ISO、拍摄时间
- **智能分组筛选**：基于EXIF拍摄时间间隔自动分组（机位→动作二级结构）
- **纯浏览器运行**：使用File System Access API，照片不上传服务器，无需安装
- **全中文界面**
