浏览器不兼容

本工具需要 File System Access API 来读取本地文件夹，您的浏览器不支持此功能。

请使用以下浏览器之一：

  • Google Chrome（推荐，版本 86+）
  • Microsoft Edge（版本 86+）
  • 其他基于 Chromium 的浏览器（如 Brave、Opera）

Firefox 和 Safari 暂不支持此功能。
